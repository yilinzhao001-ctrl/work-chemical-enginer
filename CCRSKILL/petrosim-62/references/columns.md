# 蒸馏塔参考

## 塔类型

- **蒸馏塔** (Distillation): 常压/减压精馏
- **再沸吸收塔** (Reboiled Absorber): 带再沸器的吸收塔
- **抽提蒸馏** (Extractive Distillation): 添加溶剂的蒸馏
- **反应蒸馏** (Reactive Distillation): 带化学反应的蒸馏

## 塔结构

```
      塔顶产品 ← [冷凝器] ← 第1块理论板
                              ↓
                        若干理论板
                              ↓
      塔底产品 → [再沸器] → 最后一块理论板
```

**关键术语**:
- F = 进料流
- L = 液相流股
- V = 气相流股
- LSD = 侧线液相抽出
- VSD = 侧线气相抽出
- Q = 能量流股

## 添加蒸馏塔

1. **Operations → Refining → Distillation Column** (或 F4)
2. 拖拽到流程图
3. 双击打开属性视图

## 属性视图标签

| 标签 | 内容 |
|------|------|
| Connections | 输入/输出连接 |
| Operating Data | 操作参数 (N, F, Q) |
| Design | 设计参数 (板数、进料位置) |
| Specifications | 产品规格 |
| Profiles | 温度/压力/组成剖面 |
| Column Profiles | 塔内详细剖面图 |

## 设计参数

- **理论板数**: 总板数
- **进料板位置**: 第几块板进料
- **回流比**: L/D
- **再沸比**: V/B
- **侧线抽出**: 位置和流量

## 故障排除

### 不收敛
1. 检查进料状态是否合理
2. 简化规格设定
3. 调整收敛容差
4. 良好初始估计

### 产品不合格
1. 调整回流量
2. 改变进料位置
3. 添加侧线抽出
4. 检查压力分布

### 压降异常
1. 检查塔板参数
2. 验证物性计算
3. 调整塔径

## 塔板效率

- 默认: 100% (理论板)
- 实际效率 50-80%
- 影响分离效果和温度分布

## 热量集成

- 预热链 (Pre-Heat Train)
- 侧线换热器
- 中段回流

## 方程

**Francis Weir 方程** (液相流动):
```
L_N = C * ρ * l_w * h^(3/2)
```

**气相流动**:
```
F_vap = k * sqrt(ΔP_friction)
```

其中 k 与塔径成正比。

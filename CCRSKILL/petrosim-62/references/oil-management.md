# 油品表征参考

## 概述

油品表征用于定义原油和石油产品的性质，将复杂混合物简化为虚拟组分进行模拟。

## 馏程 (Assay)

### 输入数据类型

1. **实沸点蒸馏 (TBP)**
   - 体积百分比 vs 温度
   - 最精确的表征方法

2. **恩氏蒸馏 (ASTM D86)**
   - 常用实验室数据
   - 需要转化

3. **关联馏程 (Correlated)
   - 从其他属性估算
   - 数据不足时使用

### 关键属性

| 属性 | 说明 |
|------|------|
| 密度 | API重度、相对密度 |
| 硫含量 | wt% S |
| 氮含量 | wt% N |
| 辛烷值 | RON/MON |
| 凝点 | 云点、倾点 |
| 粘度 | 运动粘度 |

## Oil Manager

### 功能
- 管理化验数据
- 输入馏程和属性
- 生成虚拟组分

### 操作步骤

1. **打开 Oil Manager**
   - Simulation Basis → Oil Manager

2. **导入化验数据**
   - 馏程数据
   - 物性数据

3. **生成虚拟组分**
   - 选择切割方案
   - 设置组分数量

4. **验证表征结果**
   - 密度曲线
   - 馏程曲线

## 虚拟组分

### 生成方法
- 窄馏分法
- 宽馏分法
- 离散法

### 参数
- 摩尔分率
- 分子量
- 密度
- 沸点
- 关键属性

## 混合 (Blend)

### 混合规则
- 线性混合
- 非线性混合 (如粘度)

### 操作
1. 选择要混合的物流
2. 指定混合比例
3. 计算混合后属性

## Property Table

### 用途
- 生成物性表
- 温度/压力扫描
- 相图绘制

## 关联式

Petro-SIM 提供多种关联式计算油品属性:

- **密度关联**: 温度压力修正
- **粘度关联**: 甘油/混合规则
- **热力学属性**: 逸度、焓、熵

## 故障排除

### 表征偏差大
- 检查化验数据
- 调整切割方案
- 验证转化模型

### 物性不连续
- 平滑化处理
- 调整虚拟组分边界

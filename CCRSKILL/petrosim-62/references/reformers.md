# 催化重整参考

## 概述

催化重整用于将石脑油转化为高辛烷值汽油组分和芳烃，同时产出氢气。

## 反应器类型

### 按再生方式分类

1. **连续再生 (Continuous Catalyst Regeneration, CCR)**
   - 催化剂在线再生
   - 连续运转
   - 转化率更高

2. **半再生 (Semi-Regenerative)**
   - 定期停机再生
   - 结构简单
   - 3-5个反应器串联

3. **不可再生 (Non-Regenerable)**
   - 催化剂不可再生
   - 一次性使用

## 单元结构

**典型配置**:
- 1-5个反应器 (带加热炉)
- 1个低压分离器
- 1个再接触器 (用于连续重整)
- 氢气循环

## 添加催化重整

1. **Operations → Refining → Catalytic Reformer**
2. 或按 F4 打开单元操作面板

## 标签页

| 标签 | 内容 |
|------|------|
| Connections | 进料/产品连接 |
| Operating Data | 操作条件 |
| Calibration | 校准参数 |
| Worksheet | 数据表 |
| Results | 结果 |

## 关键参数

### 操作参数
- **反应器入口温度** (RIT)
- **催化剂床层温度**
- **反应压力**
- **氢油比** (H/O)
- **空速** (WHSV/LHSV)

### 设计参数
- 反应器数量
- 催化剂装填量
- 反应器体积

## 校准 (REF-SIM)

使用 REF-SIM 技术手册进行校准:

1. 选择校准方法
2. 输入历史数据
3. 拟合动力学参数
4. 验证结果

### 校准参数
- 频率因子
- 活化能
- 反应级数

## 主要反应

1. **烷烃脱氢环化**: 生成芳烃和氢气
2. **烷烃异构化**: 提升辛烷值
3. **加氢裂化**: 生成轻组分
4. **积炭反应**: 副反应

## 产物分布

- 稳定汽油 (高辛烷值)
- 芳烃 (BTEX)
- 氢气 (副产)
- 干气 (C1-C2)
- 液化气 (C3-C4)

## 故障排除

### 转化率低
- 提高反应温度
- 降低空速
- 检查催化剂活性

### 积炭严重
- 提高氢油比
- 降低反应温度
- 缩短运转周期

### 产品不合格
- 调整反应条件
- 检查进料组成
- 校准模型参数
